Window

Extend from $.fn.panel.defaults. Override defaults with $.fn.window.defaults.

The window is a floated and draggable panel that can be used as an application window. By default a window can be moved, resized and closed. Its content can also be defined with either as static html or loaded dynamically via ajax.

Dependencies

  • draggable
  • resizable
  • panel

Properties

The properties extend from panel, below is the overridden and added properties for window.

Name Type Description Default
title string 窗口的标题文本。 New Window
collapsible boolean 如果定义显示可折叠按钮。 true
minimizable boolean 如果定义显示可最小化按钮。 true
maximizable boolean 如果定义显示的maximizable按钮。 true
closable boolean 如果定义显示关闭按钮。 true
closed boolean 定义如果关闭窗口。 false
zIndex number 窗口的Z指数，增加从它。 9000
draggable boolean 如果定义窗口可以拖动。 true
resizable boolean 如果定义窗口大小可以。 true
shadow boolean 如果设置为true，当窗口显示阴影会还。 true
inline boolean 定义了如何在窗口，真的呆在它的父，假去上的所有元素。 false
modal boolean 如果定义的窗口是一个模态窗口。 true
