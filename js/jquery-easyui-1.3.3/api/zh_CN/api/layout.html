Layout

Override defaults with $.fn.layout.defaults.

The layout is the container that has up to five regions: north, south, east, west and center. The center region panel is required but edge region panel is optional. Every edge region panel can be resized by dragging its border, they also can be collapsed by clicking on its collapsing trigger. The layout can be nested, thus users can build what complex layout he wants.

Dependencies

  • panel
  • resizable

Layout Options

Name Type Description Default
fit boolean 设置为true设置布局的大小适合它的父容器。创建“身体”标签的布局时，它会自动最大化整个页面全尺寸。 false

Region Panel Options

The region panel options is defined in panel component, below is some common and added properties:

Name Type Description Default
title string 布局面板标题文本。 null
region string 定义布局面板位置，该值是：北，南，东，西，中心。
border boolean 真正的显示布局面板边框。 true
split boolean 真正显示拆分条，用户可以更改面板尺寸。 false
iconCls string 一个图标显示在面板的CSS类的头一个图标。 null
href string 一个URL从远程站点的负荷数据。 null
collapsible boolean 如果定义显示可折叠按钮。 true
minWidth number 最小的面板宽度。 10
minHeight number 最低板高度。 10
maxWidth number 最大的面板宽度。 10000
maxHeight number 最大的面板的高度。 10000

Methods

Name Parameter Description
resize none 设置布局的大小。
panel region 返回指定的面板，“地区”参数的可能的值：'north’，'south”，“东”，'west’，'center”。
collapse region 指定面板的崩溃，“地区”参数的可能的值：'north’，'south”，“东”，'west”。
expand region 扩大指定面板，“地区”参数的可能的值：'north’，'south”，“东”，'west”。
add options 添加一个指定的面板，选择的参数是一个配置对象，见表面板属性的更多细节。
remove region 删除指定的面板，“地区”参数的可能的值：'north’，'south”，“东”，'west”。
