Properties

Name Type Description Default
accept selector 确定可拖动的元素会被接受 null
disabled boolean 真正的停止空投 false

Events

Name Parameters Description
onDragEnter e,source 火灾时，拖动元素被拖进。源参数表明，拖到DOM元素。
onDragOver e,source 火灾时，拖动元素被拖到。源参数表明，拖到DOM元素。
onDragLeave e,source 火灾时，拖动元素被拖走。源参数表明，拖到DOM元素。
onDrop e,source 火灾时，拖动元素掉。源参数表明，拖到DOM元素。

Methods

Name Parameter Description
options none 返回选择对象。
enable none 使可删除的功能。
disable none 禁用可删除的功能。
