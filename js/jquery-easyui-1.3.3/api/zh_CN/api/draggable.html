Properties

Name Type Description Default
proxy string,function 代理单元，用于当拖动，当设置为“克隆”，一个克隆的元素作为代理。如果一个函数被指定，它必须返回一个jQuery对象。下面的示例演示如何创建一个简单的代理对象。 null
revert boolean 如果设置为TRUE，该元素将返回到其开始位置拖动时停止。 false
cursor string CSS光标拖动时。 move
deltaX number 拖动元素的X位置对应到当前光标 null
deltaY number 拖动元素Y对应到当前光标位置 null
handle selector 手柄，开始拖动。 null
disabled boolean 真正的停止拖动。 false
edge number 在拖动宽度可以拖动。 0
axis string 定义轴拖动元素时，可用的值为“V”或“H”，当设置为null将在“V”和“H”方向。 null

Events

Name Parameters Description
onBeforeDrag e 火灾在拖动之前，返回false取消拖动。
onStartDrag e 火灾时开始拖动目标对象。
onDrag e 火灾在拖动。返回false不会拖实际上。
onStopDrag e 火灾时，将停止。

Methods

Name Parameter Description
options none 返回选择属性。
proxy none 如果代理属性设置回拖代理。
enable none 使拉动作。
disable none 禁用拖。
