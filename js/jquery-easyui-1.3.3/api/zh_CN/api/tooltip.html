Tooltip

Override defaults with $.fn.tooltip.defaults.

When a user move the mouse pointer over an element, a tooltip message window appears to display additional information. The tooltip content can contain any html elements that come from the page or via ajax.

Usage Example

Create Tooltip

1. Create tooltip from markup, add 'easyui-tooltip' class to the element, no javascript code is needed.

2. Create tooltip using javascript.

 

Properties

Name Type Description Default
position string 工具提示的位置。可能的值是：左，右”，“顶级”，底部的。 bottom
content string 工具提示的内容。 null
trackMouse boolean 真正让工具提示随着移动鼠标。 false
deltaX number 从工具提示的位置的水平距离。 0
deltaY number 从工具提示的位置的垂直距离。 0
showEvent string 事件引起了提示出现。 mouseenter
hideEvent string 事件引起了提示消失。 mouseleave
showDelay number 显示工具提示的时间延迟。 200
hideDelay number 延迟隐藏工具提示的时间。 100

Events

Name Parameters Description
onShow e 火灾时显示提示。
onHide e 火灾时隐藏工具提示。
onUpdate content 火灾时，工具提示内容更新。
onPosition left,top 火灾时，工具提示的位置改变。
onDestroy none 火灾时，工具提示被摧毁。

Methods

Name Parameter Description
options none 返回选择属性。
tip none 返回对象的尖端。
arrow none 返回箭头对象。
show e 显示工具提示。
hide e 隐藏工具提示。
update content 更新提示的内容。
reposition none 重置工具提示的位置。
destroy none 毁灭的工具提示。
