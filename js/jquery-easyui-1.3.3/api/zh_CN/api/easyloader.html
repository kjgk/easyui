EasyLoader

Usage

Load EasyUI module
	easyloader.base = '../';    // set the easyui base directory
	easyloader.load('messager', function(){        // load the specified module
		$.messager.alert('Title', 'load ok');
	});
Load script from an absolute url
	using('http://code.jquery.com/jquery-1.4.4.min.js', function(){
		// ...
	});
Load script from relative url
	// the script path relative to easyui directory
	using('./myscript.js', function(){
		// ...
	});

Properties

Name Type Description Default
modules object 预定义的模块。
locales object 预定义的地方。
base string 该easyui基目录，必须以“\/”。 The base directory will be auto setted relative to easyload.js
theme string 在“主题”目录定义的主题的名称 default
css boolean 如果定义加载CSS文件时加载的模块 true
locale string 区域名称 null
timeout number 超时值以毫秒。火灾如果超时。 2000

Events

Name Parameters Description
onProgress name 火灾在模块加载成功。
onLoad name 火灾时，模块和它的依赖成功加载。

Methods

Name Parameter Description
load module, callback 加载指定的模块。当负载成功回调函数将被调用。模块参数的有效类型：一个单一的模块名的模块阵列的CSS文件结束的。CSS的JS文件，最后的js。
