Properties

Name Type Description Default
disabled boolean 真正的禁止调整。 false
handles string 表明大小可调整的方向，n’是北，e是东，等。 n, e, s, w, ne, se, sw, nw, all
minWidth number 当调整大小的最小宽度。 10
minHeight number 当调整大小的最小高度。 10
maxWidth number 当调整大小的最大宽度。 10000
maxHeight number 最大高度时调整。 10000
edge number 边界的边缘被调整。 5

Events

Name Parameters Description
onStartResize e 火灾时开始调整。
onResize e 火灾在大小。当返回false，DOM元素不会行为实际的调整作用。
onStopResize e 一当停止调整。

Methods

Name Parameter Description
options none 返回可调整大小的选项。
enable none 使可调整大小的特征。
disable none 禁用可调整大小的特征。
