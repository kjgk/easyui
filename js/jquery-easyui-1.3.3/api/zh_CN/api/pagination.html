Pagination

Override defaults with $.fn.pagination.defaults.

The pagination allows the user to navigate data by page. It supports configurable options for page navigation and page length choices. The user can add customize buttons on right of pagination to enhance its functionality.

Dependencies

  • linkbutton

The panel on top display the first page content by default. When the user navigate the page, the 'onSelectPage' event will be triggered, in which we call 'refresh' method for the content panel to load new page content with a new URL parameter.

Properties

Name Type Description Default
total number 总的记录，应设置分页时创建。 1
pageSize number 页面大小。 10
pageNumber number 显示页码分页时创建。 1
pageList array 用户可以更改页面尺寸。pagelist属性定义的多少可以改变大小。代码示例： [10,20,30,50]
loading boolean 如果定义数据加载。 false
buttons array 定义自定义按钮，每个按钮包含两个属性：iconcls：CSS类，将显示一个背景图像处理程序：一个处理函数当按钮被单击自定义按钮可以宣布从标记： null
showPageList boolean 如果定义显示页面列表。 true
showRefresh boolean 如果定义显示的刷新按钮。 true
beforePageText string 显示输入组件在一个标签。 Page
afterPageText string 显示输入组件标签后。 of {pages}
displayMsg string 显示一个页面的信息。 Displaying {from} to {to} of {total} items
