Slider

Override defaults with $.fn.slider.defaults.

The slider enables the user to choose a numeric value from a finite range. When moving the thumb control along the track, a tip will display to represent the current value. The user can customize slider by setting its properties.

Dependencies

  • draggable

Usage Example

When using as a form field, create slider from <input> markup.

Create slider from <div/> is also allowed but the 'value' attribute is invalid.

Create slider programatically.


Properties

Name Type Description Default
width number 滑块的宽度。 auto
height number 滑块的高度。 auto
mode string 指示什么类型的滑块。可能的值：“H”（水平），V（垂直）。 h
reversed boolean 设置为true的最小值和最大值将开关位置。此属性是可用的版本以来1.3.2。 false
showTip boolean 如果定义显示值的信息提示。 false
disabled boolean 如果禁用了滑块。 false
value number 默认值。 0
min number 允许的最小值。 0
max number 最大允许值。 100
step number 增加或减少的价值。 1
rule array 显示标签旁边的滑块，|”只显示线。 []
tipFormatter function 一个函数格式滑块值。返回的字符串值将显示为尖。

Events

Name Parameters Description
onChange newValue, oldValue 火灾时的场值变化。
onSlideStart value 火灾时开始拖动滑块。
onSlideEnd value 火灾时结束拖动滑块。

Methods

Name Parameter Description
options none 将滑块选项。
destroy none 破坏滑块对象。
resize param 设置滑块尺寸。“参数”参数包含以下属性：宽度：新滑块的宽度和高度：新滑块高度
getValue none 把滑块值。
setValue value 设置滑块值。
enable none 使滑块组件。
disable none 禁用滑块组件。
