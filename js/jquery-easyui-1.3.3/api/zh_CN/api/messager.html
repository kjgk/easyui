Messager

Override defaults with $.messager.defaults.

the messager provide different styles of message boxes, including alert, confirm, prompt, progress, etc. All the messager boxes are asynchronous. Users can use the callback function to do something after interacting with messager.

Dependencies

  • window
  • linkbutton
  • progressbar

Usage

Properties

Name Type Description Default
ok string OK按钮文本。 Ok
cancel string 取消按钮的文本。 Cancel

Methods

Name Parameters Description
$.messager.show options 显示一个消息窗口在屏幕右下方的。选项参数配置对象：showtype：如何定义消息窗口将显示。有效值：零，滑动，褪色，显示。缺省为幻灯片。showspeed：定义消息窗口完成毫秒的时间。缺省值为600。宽度：定义消息窗口的宽度。缺省值为250。高度：定义消息窗口的高度。缺省值为100。标题：标题要显示面板的标题文本。味精：消息文本被显示。风格：定义消息窗口自定义dpecification风格。超时：如果定义0，消息窗口将不会关闭除非用户关闭它。定义unzero，消息窗口会自动关闭时超时。缺省值为4秒。代码示例：
